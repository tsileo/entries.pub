# entries.pub

WIP IndieWeb blog engine

## Features

 - Minimalist
   - No JavaScript
   - No admin UI (rely on Micropub)
 - Microformats aware (exports `h-feed`, `h-entry`, `h-card`, ...)
 - Micropub support
 - WebSub
   - for the Atom feed
 - Webmention support
 - Syndicate to ActivityPub via [microblog.pub](https://github.com/tsileo/microblog.pub)
